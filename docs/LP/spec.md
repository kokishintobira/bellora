# Bellora LP (MVP) Spec

## Goal
Publish a simple landing page for Bellora on https://bellora.jp with a clear concept:
"育児×仕事で忙しい30代女性向けの時短美容"

## Target
- 30代女性
- 育児と仕事を両立
- 美容に時間をかけられない
- 情報は短く、結論が早い方が好き

## Core UX
- 1ページで完結（/）
- セクションは少なく、読みやすく
- CTAは /go/[slug]（後でアフィに接続）
- PR/免責を必ず表示

## Must-have Sections
- Hero（サイト名＋一言）
- Problem/Solution（悩み→時短解決の方向性）
- Categories（朝1分 / お風呂上がり / メイク時短）
- Picks（各カテゴリ2〜3個のカード）
- FAQ（5問）
- Footer（運営方針、PR表記、免責、問い合わせメール）

## Non-goals (Not now)
- ブログ（記事量産）
- 会員機能
- 管理画面（/adminはプレースホルダーのみ）
- 画像生成やロゴ作成
